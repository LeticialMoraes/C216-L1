# C216 - L1 -> Sistemas Distribuídos

Repositório contendo exercícios, práticas e projetos desenvolvidos durante a disciplina **C216 - Sistemas Distribuídos** do **Instituto Nacional de Telecomunicações (Inatel)**.

A disciplina aborda conceitos fundamentais relacionados ao desenvolvimento de **sistemas distribuídos**, incluindo organização de sistemas, comunicação entre processos, e desenvolvimento de aplicações que utilizam múltiplos serviços ou componentes.

O repositório está organizado por práticas e exercícios desenvolvidos ao longo da disciplina.
